# ROCART Diagnostic v0.4

Application mobile-first de diagnostic pour décors thématiques (parcs d'attractions, zoos, aquariums).

## Installation

```bash
npm install
npm run dev
```

## Build pour Vercel

```bash
npm run build
npm run start
```

## Architecture

- Next.js 14 + React 18
- PWA (Progressive Web App) - fonctionne offline
- Responsive mobile-first
- Stockage local des diagnostics
- Export HTML/JSON

## Déploiement

Connecté à Vercel via GitHub. Chaque push déclenche un redéploiement automatique.
